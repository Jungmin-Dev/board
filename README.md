# board
Vue + Spring boot를 활용한 게시판 만들기

<h2>요구사항</h2>

- 회원가입(이메일 인증) 및 로그인
- 게시글 작성, 수정, 삭제, 조회
- 댓글 및 재댓글 작성 및 수정
- 첨부파일 등록


<h2> 개발 환경(IntelliJ IDEA) </h2>
<h3>front-end</h3>

- vue: 2.6.14
- vue-router: 3.5.3
- vuetify: 2.6.0
- vuex: 3.6.2
- axios: 0.27.2

<h3>back-end</h3>

- java: 11
- spring-boot: 2.7.1
- mybatis
- gradle
- database: oracle, mybatis
- lombok
